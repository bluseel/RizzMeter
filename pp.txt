git init -> initializes
git status -> see status of files
git add -> ready a file/files for stagging
git commit -m <"any comment"> -> commits all the files in stagging state (local)
git branch -M main -> -M: making root branch
git push -u origin main -> pushes all files committed to github
git branch checkout -b "Branch name" -> checkouts and switches to the new branch
